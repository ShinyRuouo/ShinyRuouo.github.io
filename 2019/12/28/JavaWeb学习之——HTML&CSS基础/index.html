JavaWeb学习之——HTML&CSS基础

长久以来，我一直自诩为Java后端开发程序员，只要碰到界面等可视化部分，我就会说

“这是前端程序员该干的事，我只提供服务。到时候我把接口文档一写，你看着调用就完事了。”

这句话其实是表明我想推卸责任…不想做太多的工作。

而且完全对前端一窍不通的话，我觉得对接的时候效率是很低的。

之前并没有系统地学习javaweb，做的一些小打小闹的项目都是用到什么学什么，比如需要用到数据库时才开始学习JDBC。那目前还是学习阶段，所以我决定从基础开始学起，先构建出一个完整的框架。
